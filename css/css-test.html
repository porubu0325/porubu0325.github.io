CSS関連用メモ

CSSで使う単位には以下の単位がある

  • px: モニタの画素（px）を1とする単位。
  • %: パーセントを使用して割合を指定できる
  • em: 親要素の大文字Mの高さ（=フォントサイズ）を1とする単位。メイリオ、ヒラギノ、Noto Sans...と、ユーザ環境によってフォントのサイズが変わってしまう環境において、その時々のフォントサイズに応じたサイズ指定ができる。
  • ex: 親要素の小文字xの高さを1とする単位
  • rem: ルート要素（html要素）の大文字Mの高さを1とする場合の単位。emとは違い、remは常に最上位のルート要素に指定されているフォントサイズを基準に計算される。例えばルートで16pxのフォントサイズが指定されているとき、li要素を1.2emに指定した場合と1.2remに指定した場合、emの方がremより大きくなる。
  • vw: viewportの幅の1/100を1とする
  • vh: viewportの高さを1/100を1とする
  • pt: ポイント。1ポイントは1/72インチ
  • pc: パイカ。1パイカは12ポイント
  • mm、cm、in: 一般的に使用されているものと同様。1インチ（in）は2.54センチメートル。
